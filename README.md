# KilljoyINK Archives

**A complete provenance record of every NFT minted by killjoyINK on the Tezos blockchain.**

Because the work deserves documentation as serious as the work itself.

## Folder Structure

```
killjoyink-archives/
├── index.html          ← Main archive & search interface
├── mint/               ← Individual mint pages (auto-generated by JS from hash routing)
├── data/               ← Place for cached mints.json snapshots (optional)
├── assets/
│   ├── style.css       ← Full design system (dark archival theme)
│   └── archive.js      ← Live data fetching + all UI logic
├── _config.yml         ← GitHub Pages config
└── README.md
```

## LIS Notes (Library & Information Science)

This archive follows best practices from digital preservation and library cataloging:

- **Dublin Core-adjacent metadata**: title, creator, date, format, identifier, description
- **TZIP-21 standard**: Tezos NFT metadata standard (analogous to schema.org for on-chain assets)
- **Persistent identifiers**: IPFS CIDs and Tezos contract+tokenId pairs as stable references
- **Provenance tracking**: block number, timestamp, first minter, contract platform
- **Exportability**: CSV for long-term data portability independent of this interface
- **Open access**: no paywalls, no login, no analytics
